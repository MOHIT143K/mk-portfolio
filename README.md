## Getting Started

### Prerequisites

- Node.js 18+
- npm 9+

### Install & Run

```bash
git clone https://github.com/MOHIT143K/mk-portfolio.git
cd mk-portfolio
npm install
npm start
```

Opens at [http://localhost:3000](http://localhost:3000).

### Build for Production

```bash
npm run build
```

Output goes to the `build/` folder, ready for static hosting (Vercel, Netlify, GitHub Pages, etc.).
